快速幂

快速幂，二进制取幂（Binary Exponentiation，也称平方法），是一个在 O(log⁡n)\Omicron \lparen \log n\rparenO(logn) 的时间内计算 ana^nan 的小技巧，而暴力的计算需要 O(n)\Omicron \lparen n\rparenO(n) 的时间。而这个技巧也常常用在非计算的场景，因为它可以应用在任何具有结合律的运算中。其中显然的是它可以应用于模意义下取幂、矩阵幂等运算。

313=3(1101)2=38⋅34⋅313^{13}=3^{(1101)_2}=3^8 \cdot 3^4 \cdot 3^1313=3(1101)2​=38⋅34⋅31

代码实现​

def bin_pow(a, b):
    res = 1
    while b > 0:
        if b & 1:
            res = res * a
        a = a * a
        b >>= 1
    return res
