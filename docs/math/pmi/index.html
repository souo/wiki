点互信息PMI (Pointwise Mutual Information)

对于随机变量 x 和 y，它们的 PMI 定义如下:

pmi⁡(x;y)≡log⁡p(x,y)p(x)p(y)=log⁡p(x∣y)p(x)=log⁡p(y∣x)p(y)\operatorname {pmi} (x;y)\equiv \log {\frac {p(x,y)}{p(x)p(y)}}=\log {\frac {p(x|y)}{p(x)}}=\log {\frac {p(y|x)}{p(y)}}pmi(x;y)≡logp(x)p(y)p(x,y)​=logp(x)p(x∣y)​=logp(y)p(y∣x)​

其中，P(x)P(x)P(x) 表示 x 发生的概率，P(y)P(y)P(y) 表示 y 发生的概率，P(x,y)P(x, y)P(x,y) 表示 x 和 y 同时发生的概率。 PMI 的值越高，表明相关性越强。
