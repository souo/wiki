监督式学习和情感分析

在监督式学习中，输入特征X和一组标签Y. 要确保根据数据能获得最准确的预测, 目标是尽可能将错误率或损失减到最少.

要进行情感分析，您首先必须将文本（即“我很高兴，因为我正在学习 NLP”）表示为特征，然后训练逻辑回归分类器，然后可以使用它对文本进行分类。
